# Contacts
源码参考自：
https://github.com/aJIEw/PhoneCallApp.git

#### 介绍
通讯录与拨号

#### 软件架构
适配安卓应用 [AIDE Pro] 的 Gradle 编译结构。
也适配安卓应用 [AndroidIDE] 的 Gradle 编译结构。


#### Gradle 编译说明
调试版编译命令 ：gradle assembleBetaDebug
阶段版编译命令 ：gradle assembleStageRelease

#### 使用说明

在安卓系统中需要设置两个权限允许。
1.自启动权限允许。
2.省电策略-无限制权限允许。
